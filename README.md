# streamlit-book-app
A simple Book Management Web Application built with Python, Streamlit, and SQLite.
This application allows users to add, view, update, delete, and search books in a local database.

## Features
- Add a new book
- View book list
- Update book information
- Delete a book
- Search books by title
- Simple and interactive web UI

## Tech Stack
- Python
- Streamlit
- SQLite
- Pandas

## How to Run
1 Install dependencies
  -pip install —r requirements.txt
2 Run the application
  -streamlit run app.py
3 Open browser
 -http://localhost:8501

## Main Functions
Add Book
 -Users can add a new book with title, author, and genre.
Book List
 -Display all books stored in the database.
Update Book
 -Users can update book information using the Book ID.
Delete Book
 -Remove a book from the database using the Book ID.
Search
-Search books by title.
